1. Make Sure appropriate credentials are configured for RabbitMQ at XmlToCSV\UploaderParserApi\appsettings.json.

2. Make sure Flask is up and python app is running. (Use below commands if not ruuing):
  a. set FLASK_APP=XmlToCSVConverter.py ('XmlToCSVConverter.py' is located at Python\Python36-32\Scripts)
  b. flask run

3. XmlToCSV\UploaderParserApi is deployed and its url is set at XmlToCSV\XmlToCSV\appsettings.json
